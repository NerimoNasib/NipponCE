# NipponCE

## How to run React Client:
1. `cd ReactClient`
2. `npm install`
3. `npm run dev`

## How to run Dotnet Client:
1. `cd DotnetClient`
2. `dotnet build`
3. `dotnet run`  
   or  
   `dotnet watch run`

## How to migrate new files to PostgreSQL:
1. `dotnet ef migrations add <commit message>`
2. `dotnet ef database update`
